Evoli #

Evoli is the web framework for Go that makes it easy to build fast, reliable and performant web applications. It lays the foundation for developers to build apps with a focus on simplicity and performance.

WARNING

Evoli is currently in 0.x status. It is not ready for production use and is subject to change at any time.
